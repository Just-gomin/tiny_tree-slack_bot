# 스파이더 카드 게임 실시간 대전 서비스 MVP 기획서

## 1. 제품 개요

**제품명:** SpiderDuel (스파이더듀얼)

**핵심 가치 제안:**
전통적인 스파이더 솔리테어를 2인 실시간 대전으로 재해석하여, 혼자 하던 카드 게임을 친구와 경쟁하며 즐기는 새로운 경험을 제공합니다.

**타겟 사용자:**

- 주 타겟: 25-45세 직장인, 점심시간이나 출퇴근 중 가볍게 즐길 수 있는 게임을 찾는 사용자
- 부 타겟: 기존 솔리테어 게임 애호가, 두뇌 게임을 선호하는 캐주얼 게이머
- 특성: 모바일 게임 경험이 있으며, 짧은 플레이 시간(5-10분) 선호, 경쟁을 즐김

## 2. 주요 기능

### 기능 1: 빠른 매칭 시스템

- **설명:**
  - 버튼 한 번으로 실시간 대전 상대와 자동 매칭
  - 평균 매칭 시간 10초 이내 목표
  - 비슷한 레벨/승률의 상대와 매칭 (±2 레벨 범위)
- **사용자 시나리오:**

    ```
    1. 사용자가 앱 실행 후 "대전 시작" 버튼 클릭
    2. 매칭 대기 화면 표시 (현재 대기 중인 유저 수 표시)
    3. 상대 발견 시 "상대를 찾았습니다!" 알림과 함께 3초 카운트다운
    4. 게임 화면으로 전환되며 동시에 게임 시작
    
    ```

- **우선순위:** P0 (핵심 기능)

### 기능 2: 동일한 카드 배치로 진행되는 대전 게임

- **설명:**
  - 양쪽 플레이어에게 완전히 동일한 카드 배치 제공
  - 실시간으로 상대방의 진행 상황 확인 (완성된 줄 수, 남은 카드 수)
  - 먼저 모든 카드를 정리하면 승리
  - 게임 중 실시간 싱크 (상대방이 카드를 움직이는 것은 보이지 않음, 진행도만 표시)
- **사용자 시나리오:**

    ```
    1. 게임 시작 시 양쪽에 동일한 10줄의 카드 배치
    2. 화면 상단에 상대방 진행 바 표시 (완성된 줄 수/10)
    3. 사용자가 카드를 드래그하여 배치
    4. 한 줄(K부터 A까지 동일 무늬) 완성 시 자동으로 제거
    5. 모든 줄을 먼저 완성하는 플레이어가 승리
    6. 승리 시 "+20 점수" 획득, 패배 시 "-10 점수"
    
    ```

- **우선순위:** P0 (핵심 기능)

### 기능 3: 게스트/회원 게임 지원

- **설명:**
  - 게스트: 앱 설치 후 즉시 게임 가능, 닉네임만 입력 (랜덤 닉네임 자동 생성 옵션)
  - 회원: 이메일 또는 소셜 로그인 (Google, Apple), 전적 및 순위 저장
  - 게스트는 10판까지 플레이 가능, 이후 회원가입 유도
- **사용자 시나리오:**

    ```
    [게스트]
    1. 앱 실행 → "게스트로 시작" 버튼 클릭
    2. 닉네임 입력 화면 (기본값: "플레이어1234" 형태의 랜덤 닉네임)
    3. 즉시 대전 가능, 우측 상단에 "로그인하고 전적 저장하기" 배너
    
    [회원]
    1. "로그인" 버튼 클릭
    2. 이메일/비밀번호 또는 소셜 로그인 선택
    3. 로그인 완료 후 프로필 화면으로 이동 (닉네임, 레벨, 승률 표시)
    
    ```

- **우선순위:** P0 (진입 장벽 최소화)

### 기능 4: 간단한 프로필 및 전적 시스템

- **설명:**
  - 닉네임, 프로필 이미지 (아바타 8종 중 선택 또는 기본 이미지)
  - 전적: 총 게임 수, 승/패, 승률, 현재 레벨
  - 레벨 시스템: 100점 = 1레벨, 최대 50레벨
  - 최근 10경기 기록 (상대 닉네임, 승패, 소요 시간)
- **사용자 시나리오:**

    ```
    1. 메인 화면에서 좌측 상단 프로필 아이콘 클릭
    2. 프로필 화면 표시
       - 상단: 프로필 이미지, 닉네임, 레벨 (예: Lv.15)
       - 중단: 통계 카드
         * 총 게임: 127판
         * 승리: 68판 (53.5%)
         * 패배: 59판
       - 하단: 최근 전적 리스트
         * "vs Player456 | 승리 | 4:23"
         * "vs Spider_Pro | 패배 | 6:12"
    
    ```

- **우선순위:** P1 (사용자 유지에 중요)

### 기능 5: 친구 대전 (방 코드 시스템)

- **설명:**
  - 6자리 방 코드 생성 및 공유
  - 방 생성자가 게임 시작 버튼 클릭 시 게임 시작
  - 방 유효 시간: 5분 (시간 내 입장 안 하면 자동 삭제)
  - 친구 대전은 점수/레벨에 영향 없음 (연습 모드)
- **사용자 시나리오:**

    ```
    1. 메인 화면에서 "친구 대전" 버튼 클릭
    2. "방 만들기" 또는 "방 참가하기" 선택
    
    [방 만들기]
    1. 6자리 코드 자동 생성 (예: "A3K9F2")
    2. 공유 버튼으로 코드 복사 또는 카카오톡 공유
    3. 친구 입장 대기 (최대 5분)
    4. 친구 입장 시 "게임 시작" 버튼 활성화
    
    [방 참가하기]
    1. 6자리 코드 입력 화면
    2. 코드 입력 후 "참가" 버튼 클릭
    3. 방 생성자의 게임 시작 대기
    
    ```

- **우선순위:** P1 (바이럴 확산에 중요)

### 기능 6: 실시간 리더보드

- **설명:**
  - 주간 TOP 100 랭킹 (매주 월요일 0시 초기화)
  - 랭킹 기준: 해당 주 획득 점수 합계
  - 내 순위 표시 (TOP 100 밖이면 "내 순위: 356위" 형태로 표시)
  - 각 랭커 클릭 시 프로필 정보 확인 가능
- **사용자 시나리오:**

    ```
    1. 메인 화면에서 "랭킹" 탭 클릭
    2. 리더보드 화면 표시
       - 상단: "이번 주 랭킹" 타이틀 + 초기화까지 남은 시간
       - 1-3위: 큰 카드 형태 (금/은/동 배지, 닉네임, 점수)
       - 4-100위: 리스트 형태 (순위, 닉네임, 레벨, 점수)
       - 하단 고정: "내 순위: 23위 | 3,450점" (배경색 강조)
    3. 랭커 닉네임 클릭 시 해당 유저 프로필 모달 표시
    
    ```

- **우선순위:** P1 (경쟁 요소 강화)

### 기능 7: 게임 내 튜토리얼

- **설명:**
  - 첫 실행 시 자동 표시 (스킵 가능)
  - 단계별 가이드: 카드 이동 → 줄 완성 → 게임 목표 설명
  - 설정에서 언제든 다시 볼 수 있음
  - 대화형 튜토리얼: 실제로 카드를 움직여보며 학습
- **사용자 시나리오:**

    ```
    1. 앱 첫 실행 시 "튜토리얼 시작" / "건너뛰기" 선택 화면
    2. 튜토리얼 선택 시:
       - Step 1: "같은 무늬의 카드를 내림차순으로 정렬하세요"
         → 화면에 간단한 카드 배치, 화살표로 이동 유도
       - Step 2: "K부터 A까지 같은 무늬로 완성하면 줄이 사라집니다"
         → 실제로 완성해보는 미션
       - Step 3: "상대보다 먼저 모든 카드를 정리하세요!"
         → 대전 화면 구성 설명
    3. 완료 후 "대전 시작" 버튼으로 자동 이동
    
    ```

- **우선순위:** P2 (사용자 온보딩 개선)

## 3. UI/UX 방향성

**디자인 테마:** Modern Minimal with Playful Accents

- 깔끔하고 직관적인 인터페이스
- 카드 게임의 클래식함 + 현대적인 대전 게임의 역동성
- 눈의 피로를 줄이는 부드러운 색감

**주요 화면 구성:**

1. **메인 화면 (홈)**
    - 상단: 프로필 아이콘(좌), 설정 아이콘(우)
    - 중앙: 큰 "대전 시작" 버튼 (펄스 애니메이션)
    - 하단: "친구 대전" / "랭킹" / "프로필" 네비게이션 바
2. **매칭 대기 화면**
    - 중앙: 스피너 애니메이션 + "상대를 찾는 중..." 텍스트
    - 하단: "대기 중인 유저: 23명" 표시
    - 취소 버튼 (하단 고정)
3. **게임 화면**
    - 최상단: 상대방 진행 바 (얇은 바 형태, 10개 세그먼트)
    - 상단: 내 점수 / 타이머 / 남은 덱 카드 수
    - 중앙: 10줄의 카드 배치 (스크롤 가능)
    - 하단: 덱 카드 더미 (터치 시 각 줄에 카드 1장씩 분배)
    - 좌측 상단: "나가기" 버튼 (경고 모달 표시)
4. **게임 결과 화면**
    - 상단: "승리!" / "패배" 대형 텍스트 (애니메이션)
    - 중앙: 통계 카드
        - 소요 시간: 4분 23초
        - 획득/손실 점수: +20점
        - 새 레벨: Lv.16 (레벨업 시 축하 애니메이션)
    - 하단: "다시 대전" / "메인으로" 버튼
5. **프로필 화면**
    - 상단: 프로필 이미지(중앙), 닉네임, 레벨 배지
    - 중단: 3개의 통계 카드 (총 게임 / 승률 / 현재 연승)
    - 하단: 최근 전적 리스트 (무한 스크롤)
    - 우측 상단: "편집" 버튼
